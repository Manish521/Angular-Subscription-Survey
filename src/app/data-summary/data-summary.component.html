Subscription Survey Results conducted for a group of 100 students


  • Do you use an Android device or an iPhone?
    • 67.3% users use an iPhone
    • 22.7% users use an Android device

  • Which music subscription application do you prefer?
    • 32.2% users like using Apple Music
    • 20% users like using Google Play Music
    • 21.8% users like using Spotify
    • 17.8% users like using Pandora
    • 8.2% users like using Amazon Music

  • Which movie and TV shows subscription application do you prefer?
    • 45.4% users like using Netflix
    • 21.3% users like using Hulu
    • 18.1% users like using Amazon Video
    • 12.8% users like using HBO NOW
    • 2.2% users like using HotStar

  • Will you be willing to pay for a news subscription channel?
    • 16.2% users said Yes
    • 83.8% users said No

  • What genre of game would you pay for?
    • 11.3% users will pay for Action
    • 19.2% users will pay for Arcade
    • 8.2% users will pay for Card
    • 13.3% users will pay for Casual
    • 30.2% users will pay for Puzzle
    • 8.2% users will pay for Sports
    • 9.6% users will pay for Racing
